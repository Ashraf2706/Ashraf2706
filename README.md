## ğŸš€ Getting Started

Each project includes its own README with specific setup instructions. Here's a quick overview:

### Finance App
```bash
cd finance-app

npm run dev
# Requires: Appwrite setup, Plaid credentials, Dwolla credentials
```

### City Event Mapper
```bash
# Backend
cd city-event-mapper/backend
go mod tidy
go run main.go

# Frontend (new terminal)
cd city-event-mapper/frontend
npm install
npm run dev
# Requires: MongoDB, Mapbox API key
```

### UMBC Campus Navigator
```bash
# Backend
cd umbc-campus-navigator/team4-section6/server
npm install
npm run dev

# Frontend (new terminal)
cd umbc-campus-navigator/team4-section6/client
npm install
npm start
# Requires: MongoDB, Google Maps API key
```

### Job Application Tracker
1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select the `job tracker` folder
4. Configure Google Sheets API key and Spreadsheet ID in the extension popup
5. Visit LinkedIn or Handshake job pages to use

### Calculator & Joke Generator
Simply open the HTML files in a web browser - no installation required!
